Código para rankear todas as 171 combinações de tipos pokémon.
Existem 18 tipos e as combinações existentes são 1 e 2.
Para 1, 18
Para 2, 153 (usar fórmula da combinação simples)
A soma resulta em 171
Para o cálculo geral ofensivo e defensivo, utilzou-se os valores a seguir

valores = {
imunidade: 0
resistencia-dupla: 0.25
resistencia-simples: 0.5
neutro: 1
vantagem-simples: 2
vantagem-dupla: 4
}

Cada uma das 171 combinações interagiu diretamente com todas as 171 combinações para o cálculo ofensivo e defensivo geral.
A métrica final de referência foi potencial ofensivo - potencial defensivo.
